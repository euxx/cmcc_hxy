# CmccHxy

CMCC he xiao yuan API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cmcc_hxy'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cmcc_hxy
